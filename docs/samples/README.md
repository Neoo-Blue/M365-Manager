# docs/samples

Anonymized example inputs and outputs. Useful for training, demos, and reference shapes when working against the tool's APIs.

| Sample | Source artifact |
|---|---|
| [`incident/`](incident/) | Full anonymized incident folder (snapshot + 3 audit JSONs + report.html). |
| [`msp-dashboard/`](msp-dashboard/) | Pre-rendered HTML dashboard for a 3-tenant portfolio. |
| [`health-output/`](health-output/) | Five scheduled health-check result JSONs (breakglass / CA / license / MFA / stale guests). |
| [`sample-plan.json`](sample-plan.json) | AI assistant `submit_plan` JSON example. |
| [`sample-session-export.json`](sample-session-export.json) | AI assistant `/export` output -- redacted chat. |
| [`preview-session.log`](preview-session.log) | Sample PREVIEW-mode audit log (JSONL). |

All UPNs / GUIDs / IPs / domains in these samples are fabricated. No real customer data.
