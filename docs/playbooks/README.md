# docs/playbooks

Ready-to-execute IR-style runbooks for specific scenarios. Each one walks through: when does this apply, what to do, expected outcome, validation, undo.

- [`compromised-account.md`](compromised-account.md) — paper companion to the incident-response tool. Use during a real compromised-account incident.
- [`mass-phishing-response.md`](mass-phishing-response.md) — coordinated phishing campaign hits multiple accounts.
- [`insider-departure.md`](insider-departure.md) — voluntary / layoff / termination offboarding with handoffs.
- [`audit-prep.md`](audit-prep.md) — annual / quarterly compliance audit evidence bundle.
- [`license-true-up.md`](license-true-up.md) — pre-renewal license efficiency review.
- [`tenant-migration.md`](tenant-migration.md) — M&A / divestiture cross-tenant move.
- [`incident-triggers.md`](incident-triggers.md) — auto-detection framework reference.
